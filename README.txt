# ソートアルゴリズムシミューレーター

## 概要
本アプリは、クイックソートのアルゴリズムを視覚的に理解するためのツールです。ユーザーが入力した数値をアニメーションでソートし、クイックソートの動作を直感的に学ぶことができます。

## 機能
- 数値のリストを入力し、クイックソートの過程をアニメーションで表示
- ステップごとのソートの進行を確認可能
- 「次のステップ」ボタンで手動操作によるソートの進行
- 「元に戻す」ボタンでリセット

## 使い方
1.数値をカンマ区切りで入力します（例: `1,3,6,7,7,54,4,43`）。
2.「生成」ボタンをクリックして数値を表示します。
3. 1度「ソートを実行する」ボタンを押します。
4.「ソートを実行する」ボタンを押すと、アニメーションでクイックソートが完了します。
5. 「元に戻す」ボタンを押すと、初期状態に戻ります。

## 技術スタック
- HTML / CSS / JavaScript
- GSAP（アニメーションライブラリ）
